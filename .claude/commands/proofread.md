# 博客文章校对与格式修正

对当前打开的博客文章进行校对和格式修正。

## 任务

读取用户指定的文章文件（或当前 IDE 打开的文件），按以下规则检查并直接修改：

### 内容校对
1. **错别字**：修正错别字（如"梳理"误写成"数列"等）
2. **语句不顺**：修正重复用词、语义不通顺的句子（如"与前缀与后缀"改为"与前缀和后缀"）
3. **逻辑错误**：检查数学/逻辑表述是否准确（如边界条件"大于"vs"大于等于"）
4. **标点符号**：修正中文标点使用错误

### 格式规范
1. **行尾空格**：正文每行末尾必须有两个空格（`  `），章节标题行、图片行、空行除外
2. **段落间距**：段落之间保持两个空行（即一个空行隔开变成两个）
3. **章节标题**：标题行（`##` 开头）后面不加空格，直接换行
4. **图片地址**：图片行（`![` 开头或 `<img` 开头）后面不需要两个空格
5. **公式格式**：保持反引号原样（`` ` `` 或 ` ``` `），不换成其他格式（如 `$...$` 或 `$$...$$`）
6. **多余空行**：删除连续超过两个空行的情况，保持最多两个空行

### 不修改的内容
- Front matter（`---` 之间的内容）
- 代码块（` ``` ` 之间的内容）
- HTML 标签
- 链接地址

## 执行步骤

1. 读取文件全部内容
2. 逐段检查上述规则
3. 直接用 Edit 或 Write 工具修改文件
4. 修改完成后，汇报所做的修改列表

## 使用方式

```
/proofread _posts/2026/03/2026-03-15-some-post.md
```

如果不指定文件，则处理当前 IDE 中打开的文件。

TIME¶

TIME

返回当前服务器时间。

可用版本：
>= 2.6.0
时间复杂度：
O(1)
返回值：
一个包含两个字符串的列表： 第一个字符串是当前时间(以 UNIX 时间戳格式表示)，而第二个字符串是当前这一秒钟已经逝去的微秒数。
redis> TIME
1) "1332395997"
2) "952581"
redis> TIME
1) "1332395997"
2) "953148"
